# Kurdlan

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

Kurdlan is a general-purpose programming language written in C++, featuring syntax entirely in Kurdish Sorani. This project aims to provide a unique programming experience by incorporating Kurdish language elements into the language design.

## Features

- Syntax completely in Kurdish Sorani
- General-purpose programming capabilities
- Written in C++ for performance and portability
- Open-source and community-driven

## Installation

### Prerequisites

- C++ compiler (e.g., GCC, Clang, MSVC)
- CMake (version 3.10 or higher)

### Building from Source

1. Clone the repository:
   ```
   git clone https://github.com/yourusername/kurdlan.git
   cd kurdlan
   ```

2. Create a build directory:
   ```
   mkdir build
   cd build
   ```

3. Configure with CMake:
   ```
   cmake ..
   ```

4. Build the project:
   ```
   make
   ```

5. Run the executable:
   ```
   ./kurdlan
   ```

## Usage

Currently, Kurdlan is in early development. The provided example prints "Hello, CMake!" to demonstrate the build process.

Future usage will include writing programs in Kurdish Sorani syntax.
